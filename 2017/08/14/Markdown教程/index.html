Markdown教程

Aug 14, 2017点击

简介：

Markdown 是一种用来写作的轻量级「标记语言」，它用简洁的语法代替排版，而不像一般我们用的字处理软件 Word 或 Pages 有大量的排版、字体设置。它使我们专心于码字，用「标记」语法，来代替常见的排版格式。例如此文从内容到格式，甚至插图，键盘就可以通通搞定了。Markdown 从写作到完成，导出格式随心所欲，你可以导出 HTML 格式的文件用来网站发布，也可以十分方便的导出 PDF 格式。

3. 层次

比如写个读书笔记，你得

第一章

  1. 第一节
  • 第二节(你不用敲 “2”，自动就有了）
    • 第一小节（推荐每层次缩进四个空格）
      • 小小节 1
      • 小小节 2
    • 第二小节
